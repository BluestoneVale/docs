### 软件管理 Brew 

* 官网 https://brew.sh/zh-cn/
* 清华大学镜像 https://mirrors.tuna.tsinghua.edu.cn/help/homebrew/
* 配置文章 https://cloud.tencent.com/developer/article/2486985

**卸载**
```bash
# 官网
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/uninstall.sh)"

# 国内镜像
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/HomebrewUninstall.sh)"
# 脚本失效
# Apple Silicon 芯片专用
sudo rm -rf /opt/homebrew
rm -rf ~/Library/Caches/Homebrew
# Intel 芯片专用
sudo rm -rf /usr/local/Cellar /usr/local/Homebrew
rm -rf /usr/local/var/homebrew
```
自动识别芯片架构（Intel/Apple Silicon）
删除 /usr/local（Intel）或 /opt/homebrew（Apple Silicon）下的所有文件
清理缓存目录 ~/Library/Caches/Homebrew

**安装**
```bash
# 官网
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 清华大学镜像
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
export HOMEBREW_INSTALL_FROM_API=1
cd WorkSpace
# 从镜像下载安装脚本并安装 Homebrew / Linuxbrew
git clone --depth=1 https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/install.git brew-install
/bin/bash brew-install/install.sh
rm -rf brew-install
# 加入启动
echo >> /Users/wuxiushi/.zprofile
echo 'eval "$(/usr/local/bin/brew shellenv)"' >> /Users/wuxiushi/.zprofile
eval "$(/usr/local/bin/brew shellenv)"
echo '# Set non-default Git remotes for Homebrew/brew and Homebrew/homebrew-core.' >> /Users/wuxiushi/.zprofile
echo 'export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"' >> /Users/wuxiushi/.profile
echo 'export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"' >> /Users/uxiushi/.zprofile
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
echo 'export HOMEBREW_API_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles/api"' >> ~/.zprofile
echo 'export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"' >> ~/.zprofile
export HOMEBREW_API_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles/api"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"
# 检查是否生效
brew config
brew install ipcalc
```
